# 线段的定义：
1. 连续3笔间如果存在重叠部分，那么连接起点和终点的线就是线段
2. 至少连续三笔中出现更高的高点、更高的低点，或者，至少连续连续三笔出现了更低的高点、更低的低点。那么连接起点和终点的线就是线段。

# 规则：
1. 开头的第一笔决定了线段的方向
2. 最少3笔，总数一定是奇数
3. 向上线段必须有向上的一笔结束，向下线段必须由向下的一笔结束

# 线段的产生
1. 当线段形成后，如果继续运行形成新的线段，但新的线段的方向和原线段的方向相反，则表示原线段结束。也可以说线段被线段破坏。

# 破坏
情况：
1. 线段被笔破坏同时被线段破话？没有被破坏
2. 线段被笔破坏，但是没有被线段破坏
## 特征序列
定义：向上线段的向下笔；向下线段的向上笔-----线段中的回调笔
特征序列合并：向上线段=>顶分型；向下线段=>底分型
## 终极版本
线段只能被线段破坏。连续 >=3 笔中出现了更高的高点、更高的低点，或者 连续 >=3 笔中出现了有更低的高点、更低的低点，那么连接起点和终点的线就是线段

# 中枢
## 定义
某级别走势类型中，被至少三个连续次级别走势类型所重叠的部分。

分型、笔、线段、中枢
线段最大，里面包括中枢，中枢里面又包括小级别的线段，或者当前级别的笔，而分型是构成笔的重要部件

上升线段中的中枢一定是起始于下，终结于下；下跌线段中的中枢一定是起始于上，终结于上。
## 画中枢
上涨线段中画中枢，在存在重叠的笔中，取高点中的低点，取低点中的高点；
下跌线段中画中枢，在存在重叠的笔中，高中取低、低中取高
